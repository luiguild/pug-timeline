# PUG Timeline
A simple static sorted timeline using PUG lang, SASS (atomic design) and Gulp

## Install
```sh
npm install
```

## Run
Using Gulp
```sh
gulp build
gulp serve
```
Using npm scripts
```sh
npm run build
npm run serve
```

## Debug Server
If you want more output, set the `DEBUG` environment variables to `*` or `gulp-live-server`.
```sh
DEBUG=gulp-live-server gulp serve
```
or
```sh
npm run serve
```
